# MIT 6.S191: Introduction to Deep Learning — Course Labs
Overview
This repository serves as a comprehensive collection of my implementations for the [MIT 6.S191: Introduction to Deep Learning](https://introtodeeplearning.com/) course. Each module explores different neural network architectures, from foundational perceptrons to advanced generative models and reinforcement learning agents.

### Lab 1: Intro to DL and Music Generation
Concepts: Neural Network fundamentals, Backpropagation, Recurrent Neural Networks (RNNs).

Highlight: Developed a Character-level LSTM to generate music in ABC notation. I focused on tuning the temperature sampling to ensure the output remained musically coherent while allowing for novelty.

Tools: PyTorch, NumPy, Music21.

### Lab 2: Facial Detection Systems
Concepts: Convolutional Neural Networks (CNNs), Facial Detection, Bias in AI.

Projected Goal: Build a CNN for facial detection and explore de-biasing techniques to improve model fairness across diverse datasets.

### Lab 3: Fine-Tune an LLM, You Must!
Concepts: Policy Gradients, Deep Q-Learning, Actor-Critic Models.

In this lab, you will fine-tune a multi-billion parameter large language model (LLM). We will go through several fundamental concepts of LLMs, including tokenization, templates, and fine-tuning. This lab provides a complete pipeline for fine-tuning a language model to generate responses in a specific style, and you will explore not only language model fine-tuning, but also ways to evaluate the performance of a language model.

You will use Liquid AI's LFM2-1.2B model as the base language model to fine-tune; Google's Gemini 2.5 as an evaluation "judge" model; and Comet ML's Opik as a framework for streamlined LLM evaluation.
